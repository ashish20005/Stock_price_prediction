Core Concepts

What makes time series data different?

Trend

Trend shows the long-term direction of a stock. If Reliance moves upward for several months, the model treats that as a persistent pattern rather than random noise.

Seasonality

Some price behavior repeats over fixed intervals, such as reactions around quarterly results, sector-wide cycles, or recurring market sentiment windows.

Stationarity

Many forecasting models work better when the statistical properties of the series stay stable over time. Differencing is often used to make data more stationary.

Lag

A lag compares today's price with previous days. Forecasting models use lagged values to learn how recent history influences the next movement.

Techniques

Models used for forecasting stock prices

ARIMA

AutoRegressive Integrated Moving Average predicts future values using past observations, differencing, and error correction. It works well for strong linear patterns.

SARIMA

Seasonal ARIMA extends ARIMA by capturing repeating cycles. It is useful when stock behavior reflects periodic market or reporting rhythms.

LSTM

Long Short-Term Memory networks are a type of recurrent neural network that can learn longer dependencies and non-linear price dynamics. They help this project stand out.
